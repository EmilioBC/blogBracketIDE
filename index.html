Installing and setting up Brackets IDE

A. Installing the IDE or software that we are going to use which is called Brackets.

1.) We will start off by clicking on this link Brackets.io.... and download the latest version.

2.) Once the file is downloaded we will just run and follow the installer of it's default settings.

3.) Before we get to the actual coding, let us first setup the Brackets environment and install additonal expansions that will aide us in making our websites easily.
Now click on the brick icon on the upper right side of your screen as shown in the image below.

4.) In the expansions page, there a lot of different expansion that you can install into your brackers environment. However in this case, we will only need one expansion which will be the "Autosave" function.

-Type "Autosave" in the search bar and click "Install".

5.) After installing the Autosave expansion, just close the expansion page and come back to the main brackets page. Then we will just make sure that it is "Enabled".

- Click "File" and make sure that the "Autosave" is checked and is enabled.

6.) Now that Brackets IDE is all setup, we can now just start coding by clicking "File" then "New"

7.) Once the new page has opened, rull of the thumb is to always saving this file first by click "File" then "Save as". In this way anything you type in this page will be automatically saved with the "Autosave" function.

8.) In saving you file, it is always a best practice to save it "index.html".

- Don not forget to always put .html at the end of the filename.

CONGRATULATIONS! You are now ready to start coding with HTML.
